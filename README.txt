QUICK START GUIDE
=================

FIRST TIME SETUP:
1. Double-click: setup.bat
2. Wait for completion (30 seconds)

INSTALLING NEW PACKAGES:
- Use: install.bat package_name
- Example: install.bat matplotlib seaborn
- This automatically updates requirements files

RUNNING CODE:
- Or activate environment: venv\Scripts\activate

THAT'S IT!

Files you'll see:
- requirements.in  = packages you want (human-readable)
- requirements.txt = exact versions (auto-generated)
- venv/ = your Python environment
